
Student Marks Analysis

This project analyzes students' marks using Python, NumPy, and Pandas. It helps to understand student performance by performing data cleaning, exploration, and visualization.

Project Files

Student.csv → Dataset of student marks

analysis.ipynb → Jupyter Notebook with complete analysis

Technologies Used

1.Python

2.Pandas

3.NumPy

Matplotlib / Seaborn (for visualization)

Key Insights

Average marks of students

Highest and lowest scores

Subject-wise performance comparison

Data visualization of marks distribution

How to Run

Download the repository

Open analysis.ipynb in Jupyter Notebook / Google Colab

Run all cells to see the analysis
